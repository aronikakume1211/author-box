Add a responsive author box or guest author box with customizable layouts, colors, and image settings. Perfect for any blog or content site.

== Description ==

The **Author Box Plugin** allows you to display a responsive author box with support for guest authors. Easily customize the layout, author image, background color, text color, and border radius. This plugin integrates seamlessly with WordPress user profiles, giving users control over their profile pictures and bio.

### Key Features
- Responsive **author box** with **3 layout options**:
  - **Layout 1:** Avatar Left  
  - **Layout 2:** Avatar Right  
  - **Layout 3:** Centered  
- **Profile Picture Upload** support for users.
- **Customizable Settings**:
  - Background color & text color.
  - Image width, height, and border-radius slider (0-100%).
- Add/Remove profile pictures directly from user profiles.

== Installation ==

1. Download the plugin as a `.zip` file or install it directly from the WordPress plugin repository.
2. Upload the folder to the `/wp-content/plugins/` directory if installing manually.
3. Activate the plugin from the **Plugins** section of your WordPress admin dashboard.
4. Go to **Settings → Author Info** to configure the options.

== Usage ==

### 1. Configure Author Box Settings:
- Navigate to **Settings → Author Info**.
- Adjust the following options:
  - **Background Color**  
  - **Text Color**  
  - **Image Width & Height**  
  - **Border Radius** (via slider)
  - **Select Layout** (Choose from three layouts)

### 2. Manage User Profile Picture:
- Go to **Users → Profile** and upload a profile picture.
- Remove the picture anytime with the **Remove Picture** button.
